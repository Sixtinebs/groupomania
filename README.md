#  Clone
```
git clone https://github.com/Sixtinebs/groupomania.git && cd groupomania
```

## Backend
```
cd Backend
```

Renommer le fichier *.env.example* en *.env* et remplire les variables 


### Installer le projet

```
npm install
```

### lancer le projet
```
npm run start
```

## frontend

Depuis le dossier racine groupomania
```
cd frontend 
```

### installer le project
```
npm install
```

### Start project
````
npm run serve
````
